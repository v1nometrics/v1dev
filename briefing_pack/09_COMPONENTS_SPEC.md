# 09 — ESPECIFICAÇÃO DE COMPONENTES (para manter consistência)

## UI primitives
- Button (primary/secondary/ghost)
- Badge (tag)
- Card
- Separator
- Tooltip
- Toggle (grid/focus mode)
- Command (⌘K)

## Content components (MDX)
- Callout (note/warn/decision)
- CodeBlock (copy)
- Figure (caption)
- Quote
- Steps (ordered)
- Metrics (lista de números)

## Layout
- Container
- PageHeader (title + subtitle + meta)
- Toc (desktop sticky)
- Footer (links + RSS)

## States
- Skeleton (loading)
- Empty state (sem posts)
- Error boundary (fallback)
